# Prathmesh Atkale - Portfolio Website

A modern, responsive portfolio website showcasing my work in AI, Data Science, and Robotics.

## 🚀 Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections with fade-in effects
- Modern dark theme with cyan accents
- Social media integration
- News/achievements section

## 🛠️ Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## 📁 Project Structure

```
prathmesh-portfolio/
├── index.html
├── style.css
├── script.js
├── README.md
└── assets/
    └── images/
        └── profile.jpg (add your image here)
```

## 🎨 Color Scheme

- Background: #0a192f (Navy)
- Secondary Background: #112240
- Text Primary: #ccd6f6
- Text Secondary: #8892b0
- Accent: #64ffda (Cyan/Turquoise)

## 🖼️ Setup Instructions

1. Add your profile picture to `assets/images/profile.jpg`
2. Update social media links in `index.html`
3. Add your resume PDF (optional)
4. Customize content as needed

## 🌐 Running Locally

### Option 1: Using Python
```bash
python3 -m http.server 8000
```
Then open: http://localhost:8000

### Option 2: Using VS Code
Install "Live Server" extension and click "Go Live"

### Option 3: Using Node.js
```bash
npx http-server
```

## 📱 Responsive Breakpoints

- Desktop: > 1024px
- Tablet: 768px - 1024px
- Mobile: < 768px
